# 🚀 快速启动指南

## 方法 1: 使用启动脚本（最简单）

```bash
cd <your-project-directory>
./start.sh
```

## 方法 2: 手动启动

```bash
cd <your-project-directory>

# 1. 激活虚拟环境
source venv/bin/activate

# 2. 启动应用
python3 app.py
```

## 启动成功标志

看到以下输出表示启动成功：
```

Note: Model will be loaded on first OCR request
💡 使用 POST /api/unload-model 可以手动释放模型内存
 * Serving Flask app 'app'
 * Debug mode: on
 * Running on http://0.0.0.0:5000
 * Running on http://127.0.0.1:5000
```

## 访问应用

启动后，在浏览器访问：
- **http://localhost:5000** （如果端口 5000 可用）
- **http://localhost:5001** （如果端口 5000 被占用，脚本会自动使用 5001）

## 停止服务

在运行服务的终端按：**`Ctrl + C`**

## 常见问题

### Q: 端口被占用怎么办？
A: 脚本会自动检测并使用备用端口（5001），并显示正确的访问地址。

### Q: 首次使用很慢？
A: 首次上传图片时会自动下载模型（~800MB），需要等待 5-15 分钟。

### Q: 如何检查服务是否运行？
```bash
curl http://localhost:5000/api/status
```
